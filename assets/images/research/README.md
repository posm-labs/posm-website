# Research Images

Place your research area images in this folder.

## Required images

| Filename                       | Used on                              |
|-------------------------------|--------------------------------------|
| `laser-stabilization.jpg`     | Landing card + detail page           |
| `cold-atoms.jpg`              | Landing card + detail page           |
| `integrated-photonics.jpg`    | Landing card + detail page           |
| `automation-fpga.jpg`         | Landing card + detail page           |

## Recommendations

- **Landing card images**: look best at 16:10 aspect ratio, ≥ 800×500px
- **Detail page images**: same file is used, displayed full-width
- Use real lab photos for best results
- Dark/moody photos work well with the site's dark theme
